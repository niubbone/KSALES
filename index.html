Azioni

Carica CSV Rendiconto

Seleziona uno o più file CSV (dml*.csv oppure dips148*.csv). Il tipo viene rilevato automaticamente dal nome file. Non processa i PDF (usare la pipeline Python per la mappatura NT→GF).

Importa Estrazione BudAnal

Seleziona il file XLS ricevuto da WKI. Il file viene archiviato in Drive (se configurato) e importato nel foglio ESTRAZIONI.

Riesegui Riconciliazione

Ricalcola firma, incassi ed estrazioni. Equivale a eseguire riconciliazioneCompleta() dal GAS editor. Circa 60 secondi.

Riconciliazione completata — i dati nel tab Riconciliazione sono aggiornati.
